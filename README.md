# Stalker-SoC-FWR-Remix

A weapon reanimation and overhaul project for **S.T.A.L.K.E.R.: Shadow of Chernobyl (SoC)** and the **RMA (Old Story Restoration)** addon.

This project is built on **FWR OLR**, tweaked to work seamlessly with vanilla SoC, and packed with weapon fixes and balancing tweaks from **Final Update (FU)**. It also includes optional weapon replacements from **EWP 1.0** to fill in any missing gaps.

---

## Features

* **Vanilla Hand Models:** Using standard SoC hand models and compatible with arm retexture mod.
* **Audio Overhaul:** Firing, holstering, and drawing sounds have been updated and mixed with classic SoC audio, replacing those old build-era sounds from OLR.
* **Balanced Gameplay:** Weapon damage, hit power, and overall feel have been rebalanced using values from *Final Update* so everything fits SoC's classic flow.
* **Easy to Port:** Designed to be easily ported and dropped into pretty much any SoC mod with minimal config editing.

---

## Optional Addons

| Addon | Description |
| --- | --- |
| **EWP Abakan & LR-300 Replacer** | Replaces the FWR AKMS with the EWP Abakan, and swaps out the vanilla-animated FWR LR-300 for the EWP version. |
| **EWP Pistols Replacer** | Replaces all FWR pistols with EWP models (since base FWR only reanimates the PM and PB). |
| **EWP Sawn-Off** | Swaps the FWR full-length double-barrel shotgun for the EWP sawn-off variant. |
| **EWP Gauss** | Replaces the vanilla Gauss Rifle with the EWP version (minus the SUSAT scope). |
| **EWP FU Misc Reanimation and Fixes** | Reanimates the bolt and fixes animations and configs for the RPG-7, RG-6, Knife, and both F1 & RGD-5 grenades. Also fixes the vanilla mismatched ammo box texture bug. |
| **EWP Unique Weapons Replacer** | Overhauls unique SoC weapons using EWP models. Adds the **VSK-94** *(RMA mod only)*, **MP-133**, and **RPK**. |
| **New Shooting Sounds** | Upgrades gunshots using assets from *Mari Weapons* and suppressed sounds from *TAZ*. |
| **Real Weapon Names** | Restores real-world weapon names across all guns, just like in early builds. |
| **Build Arm Textures** | Swaps the stock SoC arm textures for classic green build-style sleeves and gloves. |
| **NLC Icons** | Replaces UI icons with style assets from *NLC* and *AMK* (Experimental). |

---

## Installation

1. Start with a clean install of **S.T.A.L.K.E.R.: Shadow of Chernobyl** (or with vanilla **RMA**).
2. Copy the contents of the main mod's `gamedata` folder into your SoC root directory.
3. *(Optional)* Pick whichever extras you want from the `Addons` folder and overwrite the files in your `gamedata` directory.
