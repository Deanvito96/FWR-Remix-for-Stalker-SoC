# Stalker-SoC-FWR-Remix

This is a reanimation project based on FWR for OLR 2.5, adapted to play nicely with SoC

Includes optional guns from EWP 1.0 to fill in the gaps

Uses SoC Hands

Other changes:
- Shoot, holster and drawing sounds were changed to SoC counterparts instead of build ones
- Changed the damage/hit power similar to soc
